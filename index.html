📦 Livrables

    🧑‍💼 Côté Jean Samuel 🧠 (le ptit génie) — à préparer en parallèle

    1. Créer un compte Google Cloud + activer la Places API (New) et générer une clé API (crédit gratuit ~200 $/mois, large pour démarrer sur Tours).
    2. Préparer un pitch de 20 secondes : « j'ai vu que vous n'avez pas de site, je crée des sites simples pour les commerces du coin… »
    3. Définir l'offre & le prix de départ (site vitrine simple) pour pouvoir répondre au téléphone.
    4. Lire la fiche RGPD/CNIL que je fournis (prospection B2B = règles à respecter : email pro, opt-out, registre).
    ⚠️ Rappel honnêteté des données : Google fournit nom / adresse / téléphone / présence d'un site, mais jamais l'email ni le nom du contact. Ces 2 champs se remplissent à l'enrichissement (page Facebook, 1er appel) — l'outil prévoit la colonne, mais aucune donnée n'est inventée.
